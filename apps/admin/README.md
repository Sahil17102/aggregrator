# ChoiceMee Logistics Admin

React and Chakra UI admin workspace for managing ChoiceMee Logistics operations.

## Scripts

- `npm start` runs the local admin app.
- `npm run build` creates a production build.
- `npm run build:netlify` creates a Netlify-ready build.

## Brand

The admin theme mirrors the client app brand tokens: navy ink, orange primary actions, pale blue surfaces, Poppins typography, and the ChoiceMee Logistics logo from `public/brand/choicemee-logistics-logo.png`.
